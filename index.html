About Me
Hi! I'm Mike. I'm a self-taught software developer, experimental musician and DJ, and decaf iced coffee enthusiast. Check out some of my software projects below!
Projects
myTunes
Social media site for musicians. Users can post text, image, and audio, friend each other, and like and comment on posts. Built in Ruby on Rails.
Underneath
A two player board game pitting humans against a subterranean worm. Each player can only see part of the whole game board. A collaborative project with a friend. I built the back end in Ruby on Rails.
Flight Booker
A site for booking airline flights. Users can search for flights given their search parameters, select their flight and book it with multiple passengers. Built in Ruby on Rails.
Chess
A fully functional version of chess for the command line, featuring both a two player version and a one player version (against a computer making random legal moves). Built in Ruby.
